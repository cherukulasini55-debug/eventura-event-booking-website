# 🎉 eVentura – Online Event Booking Website

A modern and responsive event booking website built using **HTML, CSS, JavaScript, and Bootstrap**. The platform showcases wedding, birthday, and reception decoration collections with an interactive user interface and seamless navigation experience.

## 🚀 Features

- Responsive Design
- Bootstrap 5 Integration
- Modern Navigation Bar
- Interactive Decoration Collections
- Wedding Decoration Showcase
- Birthday Decoration Showcase
- Reception Decoration Showcase
- Smooth Navigation
- Dynamic Section Display
- Mobile-Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5

## 📸 Website Sections

### 🏠 Home
A visually appealing landing page introducing eVentura and its premium event decoration services.

### 💍 Wedding Collection
Features elegant wedding decoration themes including floral and traditional mandap setups.

### 🎂 Birthday Collection
Showcases creative birthday decoration themes with balloons, cartoon characters, jungle safari themes, and more.

### ✨ Reception Collection
Displays luxurious reception stage decorations with floral arrangements, crystal lighting, and premium backdrops.

### ℹ️ About Us
Information about eVentura's vision and commitment to creating memorable event experiences.

### 📞 Contact
Provides contact details and booking information for customers.
